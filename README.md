# TVAuto

TVAuto 是一款自动化操作的电视直播浏览器，基于 WebView 播放网络资源。

## 功能介绍

- 支持手动添加频道或批量导入频道，只需提供频道名称及网页 URL。
- 理论上支持所有含有 16:9 视频比例的网页自动播放。
- 适配多种设备，包括手机、平板以及电视。

## 注意事项

- 内置频道列表仅供学习和参考使用。
- 若需实际使用，请遵守相关法律法规，确保频道内容合法合规，自行添加频道。
