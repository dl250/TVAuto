# TVAuto

TVAuto 是一款基于 WebView 的电视直播浏览器，支持多设备（手机、平板、电视）播放网络视频资源。

## ⚙️ 功能概览

- 支持手动添加或批量导入频道，只需提供名称和 URL
- 理论兼容所有 16:9 视频网页

## 📖 快速使用

### 进入频道管理

- 遥控器：**菜单键**
- 手机/平板：点击左下角 **频道管理按钮**

### 添加频道

- 单个添加：输入频道名称和直播地址
- 批量导入：格式如下（以英文分号结尾）

  ```
  "频道名称","频道地址";
  ```

  示例：

  ```
  "CCTV-1 综合","https://tv.cctv.com/live/cctv1/";"江苏卫视","https://live.jstv.com/";
  ```

### 常用操作

- 上下滑动切换频道，长按屏幕显示/隐藏侧边按键
- 遥控器数字键切换频道，确认键刷新页面
